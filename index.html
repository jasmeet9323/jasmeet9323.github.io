Advancing Layer Surface Mesh Development

Discretizing geometry using unstructured meshes is a vital part of any simulation. In my Master's thesis project, I wrote a mesh generator which produces anisotropic surface meshes ideal for boundary layer flow simulations. These meshes act as the foundation for generating hex-dominant 3D meshes.

I worked at ANSLab, UBC to complete this thesis project. The app was made a part of the in-house mesh generator GRUMMP.

I am presenting my work at AIAA Scitech Forum 2020. The paper can be found at *give paper link*. My paper received the Best Student Paper Award - Meshing Visualization and Computational Sciences for the conference.

Basic Viewers for Geometry Data Structures

As a part of Google Summer of Code 2019 project at Computational Geometry Algorithms Library (CGAL), I developed viewers for various geometry data structures. These viewers are important to quicly visualize the result of an algorithm or to debug code. Viewers were developed for several packages in the CGAL repository including Voronoi Diagram, Periodic Triangulation, Nef Polyhedra and Arrangement 2D. Find more about my work and the code at GSoC Archive.
