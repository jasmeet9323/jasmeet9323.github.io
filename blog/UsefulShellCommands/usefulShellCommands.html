Useful Shell Commands

During my initial few months as a software developer, I came across some shell commands which I needed to look up again and again. Some of these commands were so helpful that I catalogued them on my google keep notes. In hope that they might be useful to someone else as a beginner, here they are:

  • Find file in a directory with a given name 
    find /path/to/directory -name 'fileName.abc'
  • Find file(s) with given extensions containing certain text 
    grep -r --include \*.h --include \*.cpp --include \*.c --include \*.hpp text-to-search
  • Copy file from a remote system to local system 
    scp remote_username@remote_ip:/remote/file.txt /local/directory
  • Copy file from a local system to remote system 
    scp file.txt remote_username@remote_ip:/remote/directory
  • Copy file from one remote to another 
    sscp user1@host1.com:/files/file.txt user2@host2.com:/files
